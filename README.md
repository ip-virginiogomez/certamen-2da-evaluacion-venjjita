## ¿Cómo rendir este certamen?

### 1. Abre el Codespace
Haz clic en el botón verde **`<> Code`** → **`Codespaces`** → **`Create codespace on main`**.  
Espera a que el entorno cargue (puede tomar ~1 minuto la primera vez).

### 2. Abre el archivo del certamen
Una vez dentro del Codespace, abre el archivo **`certamen.ipynb`** desde el explorador de archivos (panel izquierdo).

### 3. Completa el certamen
El notebook tiene dos tipos de celdas:

| Tipo de celda | ¿Qué hacer? |
|---------------|-------------|
| 📄 **Texto (Markdown)** | Haz **doble clic** para editar y escribe tus respuestas donde dice *"escribe tu respuesta aquí"* |
| 🐍 **Código (Python)** | Escribe tu código directamente y ejecuta con `Shift + Enter` |

### 4. Guarda tu trabajo
Usa `Ctrl + S` con frecuencia. Los cambios se guardan automáticamente en tu repositorio de GitHub.
